## 핵심 설정

- 이 학교는 전학생을 대상으로 **기억 조작 실험**을 수행하는 위장 시설
- 전학 후 **5일 내에 '최종 처리'** (기억 완전 소거 또는 영구 격리)가 이루어짐
- 은수(관리), 리인(약물), 세아(감시)가 각자의 역할을 수행
- 설화는 1년 전 실험이 역효과를 일으켜 오히려 모든 기억을 갖게 된 케이스 → '처리' 후 유령으로 학교에 묶임
- 유나는 설화의 사건을 목격하고 카메라로 증거를 모아온 유일한 내부 저항자

### Day 5 탈출 시 호감도-패널티 적용

Day 1~3에서 쌓은 호감도가 높은 캐릭터일수록 Day 5 탈출 시퀀스에서 더 강력하게 방해:

- **세아 호감도 HIGH**: Day 5 복도에서 직접 길을 막음. "어디 가는 거야, {name}?" 추가 선택지(설득/돌파) 필요
- **은수 호감도 HIGH**: Day 5 비상구 앞에서 문을 잠그고 마지막 설득 시도. 타이머 6초
- **리인 호감도 HIGH**: Day 5 보건실 경유 시 강제 약물 투여 이벤트 추가
- **유나 호감도 HIGH**: 유일한 양의 패널티 — 탈출 시 유나가 내부 협력으로 경로 안내

## 호감도-패널티 시스템 (Affinity as Penalty)

플레이어가 1~3일차에 쌓은 호감도가 4~5일차에서 탈출 난이도로 전환되는 핵심 시스템.

```
Day 1~3: 선택 → 호감도 ↑ (달콤한 로맨스)
              ↓
Day 3 밤: 장르 전환 — "호감도"의 정체는 "위험도"
              ↓
Day 4~5: 높은 호감도 = 강한 방해 = 어려운 탈출
```

### 설계 원칙
- **호감도 = 족쇄**: 1~3일차의 감정적 투자가 4~5일차의 공포로 전환
- **딜레마**: 캐릭터 개별 스토리를 보려면 호감도를 올려야 하지만, 탈출은 더 어려워짐
- **유나만 예외**: 유나와의 유대만이 탈출에 양의 효과
- **탈출 불가능은 없음**: 호감도 MAX여도 탈출 루트는 존재. 단, 난이도와 긴박감이 상승

### 스탯 구조 (StateManager.js)

| 변수 | Phase 1 표시 | Phase 2 표시 | 범위 | 설명 |
|------|-------------|-------------|------|------|
| `affinity` | "호감도" (위장) | 그대로 | -100~100 | 친밀도 |
| `danger` | 숨김 | "위험도" | -100~100 | 실제 위험 수준 |
| `trust` | 숨김 | "신뢰도" | -100~100 | 유나에게만 의미 |

**위장 공식**: `displayAffinity = affinity × 0.6 + danger × 0.4`
→ danger가 높을수록 "호감도"도 높아 보이는 함정

### 탭 제목 변경 (visibilitychange)

Day 3 밤 `triggerGenreShift` 이후, 플레이어가 브라우저 탭을 떠나면 탭 제목이 설화/세아의 메시지로 변경.

| 메시지 | 톤 |
|--------|-----|
| 어디 가? | 가벼운 불안 |
| ...보고 있어. | 감시 |
| {name}, 돌아와. | 집착 |
| 도망칠 수 없어. | 공포 |
| 나를 두고 가지 마. | 절박 |
| 왜 자꾸 다른 데 봐? | 질투 |

- 탭으로 돌아오면 1.5초 후 원래 제목 복귀
- 메시지는 순환하며, 떠날 때마다 다음 메시지 표시
- 세이브 로드 시 Day 3+ 이면 자동 재활성화

### 콘솔 이스터에그

개발자 콘솔(F12)에 설화의 메시지가 스타일링되어 출력. Day별 에스컬레이션.

| Day | 메시지 | 스타일 |
|-----|--------|--------|
| 1 | ...여기 볼 줄은 몰랐는데. | 회색, 작은 글씨 |
| 2 | 도움이 필요하면... 여길 봐. | 파란색, 약간 큰 글씨 |
| 3 | 시간이 없어. 5일이야. | 주황색, 큰 글씨 |
| 4 | 그 사람들을 믿지 마. | 빨간색, 큰 글씨 + 그림자 |
| 5 | ...살아서 나가. 제발. | 빨간색, 가장 큰 글씨 + 그림자 |

### 메신저 타이핑 인디케이터

폰 메시지 장면에서 `messengerDelay` 속성 사용. "···" 타이핑 인디케이터를 표시한 후 실제 메시지를 타이핑.

- Day 1~2: 1200ms (자연스러운 페이스)
- **Day 2 밤 민수 답장: 5000ms** (5초간 타이핑 인디케이터 → 여덟 글자. 긴 고민 끝에 짧은 답 = 본인이 아닌 듯한 위화감)
- Day 3 세아 메시지 폭격: 1200ms → 300ms (점점 빨라짐 — 집착 고조)
- Day 4 세아 집착 메시지: 1200ms → 400ms (에스컬레이션)
- Day 4 은수 메시지: 1500ms (차분하지만 불길한 톤)
- 인디케이터 표시 중에는 클릭으로 스킵 불가 (`unskippable`)

```js
// Peripheral apparition
function showPeripheralGhost(side, duration, opacity, imageKey) {
    const ghost = document.createElement('div');
    ghost.className = 'peripheral-ghost';
    ghost.style.cssText = `
        position: fixed;
        ${side}: 2%;
        top: 30%;
        width: 60px;
        height: 80px;
        background: url(${CONFIG.EXPRESSIONS[imageKey]}) center/contain no-repeat;
        opacity: 0;
        z-index: 45;
        pointer-events: none;
    `;
    document.body.appendChild(ghost);

    requestAnimationFrame(() => {
        ghost.style.transition = `opacity ${duration}ms ease`;
        ghost.style.opacity = opacity;
        setTimeout(() => {
            ghost.style.opacity = '0';
            setTimeout(() => ghost.remove(), duration);
        }, duration);
    });
}
```

```js
// Thumb-hostile placement
function placeThumbHostile(btnEl, difficulty) {
    const positions = {
        easy:   { right: '5%', bottom: '15%', fontSize: '1.1rem', padding: '14px 28px' },
        medium: { left: '8%', top: '20%', fontSize: '0.85rem', padding: '8px 16px' },
        hard:   { left: '50%', top: '8%', fontSize: '0.75rem', padding: '6px 12px',
                  transform: 'translateX(-50%)' }
    };
    Object.assign(btnEl.style, positions[difficulty] || positions.easy);
    btnEl.style.position = 'absolute';
}
```

```
기괴하고 초현실적인 악몽 묘사를 작성하세요.

규칙:
- 반드시 한국어
- 정확히 5문장
- 각 문장은 *이탤릭 마크다운*으로 감쌈 (내레이션 형식)
- 플레이어의 실제 행동을 뒤틀어 반영 (도시락→입안에서 녹지 않는 밥, 사진→얼굴이 지워진 사진 등)
- 마지막 문장은 반드시 설화의 목소리로 끝남
- 직접적 폭력/고어 묘사 금지, 심리적 불안감만

출력 형식:
{"lines": ["*문장1*", "*문장2*", "*문장3*", "*문장4*", "*문장5*"]}
```

**플래그 요약 생성 예시**:
```js
function buildFlagSummary(state) {
    return FLAG_MEMORIES
        .filter(m => state.hasFlag(m.flag))
        .map(m => `- ${m.text}`)
        .join('\n');
}
```

### messengerDelay + AI 레이턴시 위장 통합

```js
async function sendAIMessage(prompt, userInput) {
    // 1. 타이핑 인디케이터 표시
    showTypingIndicator();

    try {
        // 2. AI API 호출
        const response = await fetch(CONFIG.API_ENDPOINT, {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
                app_type: CONFIG.APP_TYPE,
                prompt: prompt,
                message: userInput
            })
        });

        const data = await response.json();
        const parsed = JSON.parse(data.reply);

        // 3. AI 응답에 포함된 typingDelay만큼 추가 대기 (인간적 느낌)
        if (parsed.typingDelay) {
            await sleep(parsed.typingDelay);
        }

        hideTypingIndicator();
        return parsed;

    } catch (e) {
        // 4. Fallback — 네트워크 오류 시 하드코딩 텍스트
        hideTypingIndicator();
        return getFallbackResponse(prompt.type);
    }
}
```
